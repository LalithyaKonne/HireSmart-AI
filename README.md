# 🚀 HireSmart AI

## AI-Powered Resume Analyzer & Mock Interview Preparation Platform

HireSmart AI is a Full Stack MERN application that helps students improve their resumes and prepare for placement interviews.

---

## 🌐 Live Demo

**Frontend**

https://hire-smart-ai-eight.vercel.app

**Backend API**

https://hiresmart-ai-backend.onrender.com

---

## ✨ Features

### 🔐 Authentication

- User Registration
- Secure Login
- JWT Authentication
- Protected Routes

### 📄 Resume Analyzer

- Upload Resume (PDF)
- ATS Score
- Skill Detection
- Missing Skills
- Resume Suggestions
- Resume History

### 🎤 Mock Interview

- Java Questions
- MERN Questions
- Frontend Questions
- DSA Questions
- HR Questions
- Interview Feedback

### 📊 Dashboard

- Personalized Welcome
- Resume Analyzer
- Mock Interview
- Resume History

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/LalithyaKonne/HireSmart-AI.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
MONGO_URI=Your MongoDB URI
JWT_SECRET=Your Secret Key
PORT=5000
```
